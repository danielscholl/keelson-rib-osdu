The `@keelson/rib-osdu` package is the OSDU CIMPL bridge as a
[Keelson](https://danielscholl.github.io/keelson/) **rib**: a discovery-based
extension that contributes deterministic workflows whose structured output drives
live canvas views. The harness stays domain-free; all OSDU and cluster knowledge
lives here.

These docs mirror the keelson documentation tiers. Keelson documents the generic,
reusable rib contract; this site documents the concrete OSDU rib that implements
it.
